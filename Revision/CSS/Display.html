1. display: block

Characteristics: Starts on a new line. Takes up the full width available. Respects width and height properties.

2. display: inline

Characteristics: Does NOT start on a new line. Sits side-by-side. Ignores width and height properties (notice the size of the boxes matches only the text content).

3. display: inline-block

Characteristics: Best of both worlds. Sits side-by-side on the same line like inline, but respects width and height properties like block.
